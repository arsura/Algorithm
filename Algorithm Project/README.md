# Hill Climbing
Algorithm Project <br/>

ขอเกริ่นก่อนว่านำ Hill Climbing มาใช้หาคำตอบ (วิถีทางไปสู่คำตอบ) ของเกม 8-Puzzle โดยผู้ใช้งานเป็นผู้กำหนด สถานะเริ่มต้นของปัญหาของและสถานะสิ้นสุดที่ต้องการ โดยจากการทดลองทำหลาย ๆ สถานะที่ต้องการหาคำตอบ พบว่าบางปัญหาก็ไม่สามารถหาวิถีทางไปสู่คำตอบได้ จึงไม่การันตีว่า Hill Climbing จะสามารถแก้ปัญหา 8-Puzzle ได้ทุกรูปแบบที่ต้องการ (สรุปว่า Hill Climbing ไม่เหมาะที่จะนำมาใช้แก้ปัญหา 8-Puzzle สักเท่าไหร่) 

--------------------------------------------------------------

**คำถาม**

เมื่อปีนเขาไปติดอยู่ที่ local optimum ไปไม่ถึง global optimum ต้องใช้วิธีอะไรแก้ ?
- ใช้วิธีการ Simulated Annealing (การจำลองการอบเหนียว) 
- ใช้แนวคิดที่รับ heuristic ที่แย่บ้างเพื่อให้หลุดออกจาก local optimum
